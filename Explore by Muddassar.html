def download(web_address):

    open('2008.csv', 'wb').write(unzip)

What is the structure of your dataset?¶

Dataset includes basically 3 types of data quantitative, qualitative and categorical.
- quantitative: Distance, Delay etc.
- qualitative: UniqueCarrier, Origin, Destination etc.
- categorical: Diverted, CancellationCode, Cancelled etc.
data includes '2389217 rows and 29 columns. Data is for 2008 USA.

What is/are the main feature(s) of interest in your dataset?¶

most interesting features are delays and cancellations based on which I want to figure out performance of airlines

What features in the dataset do you think will help support your investigation into your feature(s) of interest?¶

As mentioned earlier, features like CarrierDelay, CancellationCode, Cancelled, Distance, and CarrierDelay
Based on features undermentioned parameters will be used as criteria for assessment.

  • flights cancelled/ hundred flights schedulled
  • Carrier delay (min)/ hundred flights operated
  • Carrier delay (min)/ hundred miles distance

For Bi Variate Analysis we'd have to create new columns as parameters, pairs of variables will be assessed in this sections. New columns in the new dataframe would be like:

  • column 'carrier_name'=names of the carriers
  • column 'num_schedulled'=total flights schedulled
  • column 'num_cancelled'=total flights cancelled
  • column 'num_delayed'=total flights delayed
  • column 'total_delay'=total delay in minutes

  • column 'total_distance'=total distance in miles covered


    calculated column would be like:

  • column 'pct_cancelled'=flights cancelled per 100 flights schedulled
  • column 'pct_delayed'=flights delayed per 100 flights operated
  • column 'delay_per_flight'=carrier delay in minutes per flight operated
  • column 'delay_per_mile'=carrier delay in minutes per 100 miles distance
